# Wellington

A VIM environment.

Clone the project into your `~/.vim` directory.

`git clone git@github.com:brianp/wellington.git ~/.vim/ -r`

Then symlink the `~/vimrc/` file in your home directory.
`ln -s ~/.vim/vimrc/ ~/.vimrc`

## Plugins
- Ack
- Ctrlp
- NerdCommenter
- NerdTree
- Syntastic
- ZoomWin
- DetectIndent
